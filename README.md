## Scripts

| Script     | Description                    |
| ---------- | ------------------------------ |
| `start`    | Start the application          |
| `android`  | Run the application on Android |
| `ios`      | Run the application on iOS     |
| `web`      | Run the application on web     |
| `lint`     | Lint the code                  |
| `lint:fix` | Lint and fix the code          |
| `doctor`   | Run the doctor command         |
